Elements definitions

h1
"Header 1"
Creates a large main header. Example <h1> header </h1>
h2
"Header 2"
Creates a sub header.Example <h2> subheader </h2>
h3
"Header 3"
Creates a sub-sub header. Example <h3> sub sub header </h3>
p
"Paragraph"
Creates a paragraph for text. Example <p> text </p>
ul
"Unordered List"
Creates a bulleted list without numbers. Example <ul> unordered list </ul>
ol
"Ordered List"
Creates a numbered list for items. Example <ol> ordered list </ol>
li
"Listed Item"
Adds an item to a list. Example <li> list item </li>
