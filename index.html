How to create an HTML document

Creating the document

Things you'll need:

  • Computer
  • Text Editor
  • An Internet Browser for testing

Instructions

  1. First, turn on your computer.
  2. Startup your favorite text editing program.
  3. Write all the HTML required for your web page.
  4. Save your document when you are done.
  5. To test it, simply click and drag the .html file into your favorite browser
